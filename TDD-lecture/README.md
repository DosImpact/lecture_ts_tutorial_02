## 인프런 TDD, 따라하며 배우는 TDD 개발

TDD 의 목적

- 더 안정적인 어플리케이션 개발을 위해

TDD로 얻는 이점

- 1. 디버깅 시간 단축, 유닛테스트에서 특정버그를 검출할 수 있다.
- 2. 테스트 코드와 함께 작성된 어플리케이션이 되므로, 안정적인 App이 된다.
- 3. 재설계 시간의 단축 등

TDD 앱 개발

1. 단위 테스트 작성
2. 코드 작성
3. 통합 테스트 작성

사용하는 모듈

1. JEST : 단위 테스트 모듈
2. node-mocks-http : 단위 테스트 모듈
3. supertest : 통합 테스트 위한 모듈
