# 소개 및 타입스크립트란?

ts는 js의 슈퍼셋 언어이다.  

- babel, tsc 트랜스파일러에 의해 변환이 가능하다.
- js프로젝트에 점진적으로 ts도입할 수 있다. 두 파일이 공존할 수 있음.
- JS ES6 보다 OOP에 필요한 문법들을 많이 제공해준다. 

ts playground
- 컴파일러 옵션 체크 실험 가능
- 타입스크립트 문법 피쳐들, 예제 코드 잘 정리되어 있음.
