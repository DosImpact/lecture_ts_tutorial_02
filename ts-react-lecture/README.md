- 유명한 로더는 두가지 : ts-loader, awesome-typescript-loader
- 차이점:

1. atl babel과 함께 쓸 수 있다.
2. atl은 로더가 빠르다.

- 라이브러리와 @types/파일간의 버전은 맞추는 것이 좋음
